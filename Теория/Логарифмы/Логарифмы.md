**Логарифм** - степень, в которую нужно возвести число $a$, что бы получить число $b$

$$\log_a b = x \iff a^x = b$$

---
## **Анатомия**
$$x = \log_{a} b$$
- $x$ - значение логарифма
- $a$ - основание логарифма
- $b$ - подлогарифмическое выражение (аргумент)

**Ограничения:**
$$\begin{cases}
  b > 0 \\
  a > 0 \\
  a \not = 1
\end{cases}$$

---
## **Специальные логарифмы**

- Десятичный логарифм: $\log_{10} b = \lg b$
- Натуральный логарифм: $\log_{e} b = \ln b$
	- $e = 2.71 8$  - экспонента, число Эйлера

---
## Свойства логарифмов

| Операция            | Пример                                               |
| ------------------- | ---------------------------------------------------- |
| *Сложение*          | ==$\log_{a} x + \log_{a} y = log_{a}(x \cdot y)$==   |
| *Вычитание*         | ==$\log_{a} x - \log_{a} y = \log_{a} \frac{x}{y}$== |
| *Степень аргумента* | ==$\log_{a} b^n = n \cdot \log_{a} b$==              |
| *Степень основания* | ==$log_{a^n} b = \frac{1}{n} \cdot \log_{a} b$==     |

**Переход к новому основанию:**
$$\log_{a} b = \frac{log_{n} b}{\log_{n} a}$$
**Ограничения:**
$$\begin{cases}
n > 0 \\
n \not = 1
\end{cases}$$

---
## Логарифмическая функция
$$f(x) = \log_{a} x$$
- Если $a > 1$ функция возрастает. Чем больше x - тем больше $\log_{a} x$

```tikz
\begin{document}
\begin{tikzpicture}[domain=0.1:8]

\draw[very thin,color=gray] (-0.1,-3.1) grid (8.1,3.1);

\draw[->] (-0.2,0) -- (8.2,0) node[right] {$x$};
\draw[->] (0,-3.2) -- (0,3.2) node[above] {$f(x)$};

\draw[color=red] plot (\x,{ln(\x)/ln(2)}) node[right] {$f(x)=\log_{2}x$};

\fill (1,0) circle (2pt);
\fill (2,1) circle (2pt);
\fill (4,2) circle (2pt);

\end{tikzpicture}
\end{document}

```

- Если $0 < a < 1$ функция убывает. Чем больше x - тем меньше $log_{a} x$
```tikz
\begin{document}
\begin{tikzpicture}[domain=0.1:8]

\draw[very thin,color=gray] (-0.1,-3.1) grid (8.1,3.1);

\draw[->] (-0.2,0) -- (8.2,0) node[right] {$x$};
\draw[->] (0,-3.2) -- (0,3.2) node[above] {$f(x)$};

\draw[color=red] plot (\x,{ln(\x)/ln(0.5)}) node[right] {$f(x)=\log_{0.5}x$};

\fill (1,0) circle (2pt);
\fill (0.5,1) circle (2pt);
\fill (2,-1) circle (2pt);

\end{tikzpicture}
\end{document}

```
